How to install
===================================

1. Create a Canvas in the hierarchy.
2. Add a Panel in the Canvas.
3. Right-click on the generated Panel and select UI / Dynamic V Scroll View from the menu.
4. After a while, the scroll view will automatically be created.
5. Execute.

Please refer to Example-01 Scene in the installed folder of the example project.
